# mp-objects

Popular Pure Data objects by Miller Puckette updated for 64-bit Windows,
Intel Mac, and ARM Mac.

## Objects

- `sigmund~`: sinusoidal analysis and pitch tracking
- `fiddle~`: pitch estimator and sinusoidal peak finder
